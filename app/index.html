About Kerm.is

Kerm.is is a website made by Bart Van Itterbeeck & Robbert Wolfs. These two Web&UX students wanted to do something with new web technology like WebGL, Three.js and socket.io. So they started Kerm.is, a website where you can play all kind of different carnival games.

They wanted to let the user expierence something totally new! What if the user can play a game with their smartphone as controller or just with their hands? To accomplisch this Bart & Robbert used Node.js, websockets and a leap motion.

But why don’t they use a 3D program like Unity? Because Bart & Robbert don’t like to install plugins and they wanted to learn new native, fancy, cool webtechnologies and not an old, dirty, ... language like C#. This would be a challenge they both looked forward to!

Quick, go and try these awesome games and be amazed with what the web can do these days!
